이용약관

최종 업데이트: 2026년 4월 26일

요약: MindFlow는 무료·오픈 소스로 제공되는 개인용 도구입니다. "있는 그대로(AS IS)" 제공되며 데이터 손실·서비스 중단 등에 대한 책임을 지지 않습니다. 사용자가 본인의 데이터를 직접 백업·관리할 책임이 있습니다.

1. 서비스 소개

MindFlow는 마인드맵, 타임블록(시간표 계획), 마크다운 메모 기능을 제공하는 정적 웹 애플리케이션입니다. 본 약관은 사용자가 서비스를 이용함에 있어 권리와 의무, 책임 사항을 규정합니다.

2. 무료 제공

본 서비스는 개인 사용 목적으로 무료 제공됩니다. 광고나 결제 시스템이 없으며, 향후에도 본 서비스의 핵심 기능에 대한 유료화를 계획하고 있지 않습니다.

3. 오픈 소스

본 서비스의 소스 코드는 GitHub에서 공개되어 있으며 MIT 라이선스에 따라 자유롭게 사용·수정·배포할 수 있습니다.

4. 사용자의 책임

  • 본인의 계정·기기 보안과 데이터는 본인이 관리합니다
  • 중요한 데이터는 별도로 백업하시는 것을 권장합니다 (앱 내 JSON 내보내기 기능 제공)
  • Google Drive 등 외부 서비스 연동 시 해당 서비스의 약관·정책을 따릅니다
  • 법령에 위반되는 콘텐츠를 본 서비스에 작성·저장해서는 안 됩니다

5. 면책 조항

본 서비스는 "있는 그대로(AS IS)" 제공되며, 명시적·묵시적 어떠한 보증도 제공하지 않습니다. 다음 사항에 대해 책임을 지지 않습니다:

  • 버그·결함으로 인한 데이터 손실 또는 손상
  • 서비스의 일시적 또는 영구적 중단
  • 제3자 서비스(Google Drive, GitHub 등)의 장애나 정책 변경
  • 호환성·정확성·완결성·신뢰성에 대한 보증
  • 사용자가 본 서비스를 통해 얻은 정보의 활용으로 인한 손해

6. 외부 서비스 연동

본 서비스는 다음 외부 서비스와 선택적으로 연동될 수 있으며, 각 서비스의 이용약관과 개인정보 정책은 해당 제공자의 정책을 따릅니다:

  • GitHub Pages — 정적 호스팅
  • Google Drive — 데이터 동기화 (사용자 선택)
  • GitHub Gist — 데이터 동기화 (사용자 선택)

7. 약관 변경

본 약관은 변경될 수 있으며, 변경 시 본 페이지의 "최종 업데이트" 날짜가 갱신됩니다. 중대한 변경의 경우 앱 내에서 공지합니다.

8. 서비스 종료

사용자는 언제든지 별도의 절차 없이 서비스 사용을 중단할 수 있습니다. 데이터 삭제는 브라우저 설정 또는 Google Drive에서 직접 수행할 수 있습니다.

제공자는 본 서비스를 사전 통지 후 종료할 수 있으며, 종료 시 사용자가 데이터를 내보낼 수 있는 충분한 기간을 둡니다.

9. 지적 재산권

본 서비스의 소스 코드는 MIT 라이선스에 따릅니다. 사용자가 본 서비스를 통해 작성·생성한 콘텐츠(메모·마인드맵 등)의 저작권은 사용자에게 귀속됩니다.

10. 준거법

본 약관은 대한민국 법률에 따라 해석되고 시행됩니다. 본 서비스 이용과 관련한 분쟁이 발생할 경우, 사용자의 주소지 관할 법원을 우선합니다.
