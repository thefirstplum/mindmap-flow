G
Google Drive 연결되지 않음
메모·마인드맵·이미지를 Drive 폴더에 자동 동기화 (모든 기기)
📖 OAuth Client ID 만드는 법 (10분)
1. console.cloud.google.com 접속
2. 프로젝트 만들기 → 이름: MindFlow
3. 좌측 메뉴 → API 및 서비스 → 라이브러리 → "Google Drive API" 검색 → 사용 설정
4. API 및 서비스 → OAuth 동의 화면:
   • User Type: 외부(External) 선택
   • 앱 이름: MindFlow / 사용자 지원 이메일·개발자 이메일에 본인 메일 입력
   • 범위(Scopes)는 비워두고 저장
   • 테스트 사용자에 본인 이메일 추가 (중요!)
5. API 및 서비스 → 사용자 인증 정보 → 사용자 인증 정보 만들기 → OAuth 클라이언트 ID:
   • 애플리케이션 유형: 웹 애플리케이션
   • 승인된 JavaScript 원본: https://thefirstplum.github.io 추가
   • 로컬 테스트 시: http://localhost:8000 도 추가
   • 만들기 → 클라이언트 ID 복사
6. 위 입력란에 붙여넣고 [🔗 Google 연결] → 첫 로그인 시 "확인되지 않은 앱" 경고 → 고급 → 진행 (본인 앱이라 안전)

💡 다른 기기에서도 같은 Client ID + 같은 Google 계정으로 로그인하면 자동 동기화
💡 앱이 만든 MindFlow 폴더만 접근 (drive.file 스코프 — 안전)
💡 이미지 업로드는 MindFlow/assets/에 자동 저장

☁
연결되지 않음
GitHub Token으로 모든 기기에서 자동 동기화 (iOS·Mac 모두)
📖 GitHub Token 만드는 법 (3분)
1. github.com/settings/personal-access-tokens/new 접속
2. Token name: MindFlow Sync
3. Expiration: 원하는 기간 (최대 1년 권장)
4. Repository access: Public Repositories (read-only) 그대로
5. 아래로 스크롤 → Account permissions → Gists 항목 → Read and write 선택
6. 맨 아래 Generate token → github_pat_xxx 복사 → 위 입력란에 붙여넣고 [연결]

💡 다른 기기에서 동일하게 동작하려면: 같은 Token + 같은 Gist ID 입력
💡 Gist ID는 처음 연결 시 자동 생성되어 표시됩니다 — 다른 기기 입력란에 복사해 주세요
💡 Token은 이 브라우저 localStorage에만 저장됩니다 (외부 전송 안 됨)
