G

💡 본인 Google 계정으로 로그인 → MindFlow 폴더에 자동 저장 · 이미지는 assets/에 (drive.file 스코프, 본 앱이 만든 파일만 접근)
⚙️ 고급: 본인 OAuth Client ID 사용
기본 Client ID로 자동 연결됩니다. fork해서 자신의 Cloud 프로젝트로 쓰려면 아래 입력란에 직접 붙여넣고 [연결] 누르세요.
▸ 다른 동기화 옵션 (Gist · 로컬 폴더 · JSON)
☁
연결되지 않음
GitHub Token으로 모든 기기에서 자동 동기화 (iOS·Mac 모두)
📖 GitHub Token 만드는 법 (3분)
1. github.com/settings/personal-access-tokens/new 접속
2. Token name: MindFlow Sync
3. Expiration: 원하는 기간 (최대 1년 권장)
4. Repository access: Public Repositories (read-only) 그대로
5. 아래로 스크롤 → Account permissions → Gists 항목 → Read and write 선택
6. 맨 아래 Generate token → github_pat_xxx 복사 → 위 입력란에 붙여넣고 [연결]

💡 다른 기기에서 동일하게 동작하려면: 같은 Token + 같은 Gist ID 입력
💡 Gist ID는 처음 연결 시 자동 생성되어 표시됩니다 — 다른 기기 입력란에 복사해 주세요
💡 Token은 이 브라우저 localStorage에만 저장됩니다 (외부 전송 안 됨)
📁
폴더가 연결되지 않음
폴더를 선택하면 .md 파일을 자동으로 동기화합니다
💡 옵시디언 vault나 iCloud Drive 폴더를 선택하면 그곳의 .md 파일을 메모로 불러오고, 변경사항이 실시간으로 그 폴더에 저장됩니다
💡 데스크톱 Chrome / Edge / Brave에서 동작합니다 (Safari·iOS 미지원)
💡 외부에서 .md 파일을 수정한 경우 "다시 불러오기"로 동기화하세요
💡 iOS에서는 "공유" → "파일에 저장" → iCloud Drive 사용
⚠️ JSON 불러오기는 현재 데이터를 덮어씁니다
