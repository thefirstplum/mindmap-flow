개인정보 처리방침

최종 업데이트: 2026년 4월 26일

요약: MindFlow는 사용자의 메모·마인드맵·일정 데이터를 사용자의 브라우저(localStorage)에 저장합니다. 사용자가 직접 연결한 Google Drive 외에는 어떠한 외부 서버로도 데이터가 전송되지 않습니다. 회원가입·광고·추적이 없습니다.

1. 수집하는 정보

MindFlow는 별도의 회원가입을 요구하지 않으며, 다음 정보를 수집하지 않습니다:

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • IP 주소, 위치 정보, 기기 식별자
  • 광고 추적, 분석 쿠키, 사용 통계

2. 사용자 데이터의 처리

사용자가 앱에서 작성하는 모든 데이터(메모, 마인드맵, 일정, 이미지)는 다음 위치에 저장됩니다:

  • 브라우저의 localStorage — 해당 기기의 브라우저 안에만 저장 (오프라인 사용 가능)
  • 사용자 본인의 Google Drive (선택적) — 사용자가 동기화를 활성화한 경우 자동 생성된 MindFlow 폴더 내

개발자, 호스팅 제공자(GitHub Pages), 또는 제3자가 사용자 데이터에 접근할 수 없습니다.

3. Google Drive 연동

사용자가 Google Drive 동기화를 활성화한 경우:

  • OAuth 2.0 표준을 통해 사용자가 직접 권한을 부여합니다
  • 최소 권한 스코프 drive.file 사용 — 본 앱이 만든 파일만 접근 가능. 사용자의 다른 Drive 파일·폴더는 접근하지 않습니다
  • 액세스 토큰은 사용자의 브라우저 메모리에만 일시적으로 저장되며, 외부로 전송되지 않습니다
  • 사용자는 언제든 Google 계정 → 보안 → 제3자 앱에서 권한을 취소할 수 있습니다

4. 정보 공유

MindFlow는 사용자의 데이터를 어떤 제3자와도 공유하지 않습니다. Google Drive에 저장된 데이터는 Google의 개인정보 정책을 따릅니다.

5. 데이터 보관 및 삭제

사용자는 언제든지 다음 방법으로 모든 데이터를 삭제할 수 있습니다:

  • 앱 내에서 개별 항목 삭제 또는 동기화 모달의 "연결 해제"
  • 브라우저의 localStorage 삭제 (브라우저 설정 → 사이트 데이터 지우기)
  • Google Drive에서 MindFlow 폴더 직접 삭제

6. 보안

MindFlow는 정적 웹 애플리케이션으로 자체 서버를 운영하지 않습니다. GitHub Pages를 통해 HTTPS로 제공되며, Google Drive와의 통신도 모두 HTTPS로 암호화됩니다. 사용자의 기기 보안과 Google 계정 보안은 사용자가 직접 관리해야 합니다.

7. 어린이 개인정보

본 서비스는 만 14세 미만 어린이로부터 의도적으로 정보를 수집하지 않습니다. 어린이가 본 서비스를 사용한 정황이 발견되면 즉시 보호자에게 통지하고 관련 데이터를 삭제하도록 안내합니다.

8. 정책 변경

본 정책은 변경될 수 있으며, 변경 시 본 페이지의 "최종 업데이트" 날짜가 갱신됩니다. 중대한 변경의 경우 앱 내에서 공지합니다.
